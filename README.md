# minigit - yet another git implementation

## INSTALL

make

## How to use
```
minigit cat-file -s objectfile
minigit cat-file -t objectfile
minigit cat-file -p objectfile
```

objectfile can be a commit, tree , or blob.

## Run test

```
prove
```
